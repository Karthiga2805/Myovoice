# Sensor Data Acquisition and Preprocessing Pipeline

This folder contains modules for sensor calibration, data collection, and preprocessing pipelines designed for machine learning model training.

## Directory Structure

* **Data collection/**: Scripts for interfacing with various sensor configurations to acquire raw dataset logs.
* **Sensor Calibration/**: Utilities for calibrating hardware sensors to ensure data accuracy and baseline consistency.
* **Helper scripts/**: Preprocessing tools used to convert recorded `.csv` data files into `.cpp` source files for integration into the model training pipeline.
